## Installation

js-library-boilerplate is available on npm/yarn:

```bash
$ npm install js-library-boilerplate --save
$ yarn add js-library-boilerplate
```

## Usage

### With ES6/import

```js
import { sum } from 'js-library-boilerplate';

sum(2, 2); // => 4
```

### With require

```js
const sum = require('js-library-boilerplate').sum;

sum(2, 2); // => 4
```
